# React + Express Fullstack Starter for Frontend Intern Task

## What you get
- React (Vite) frontend with Tailwind CSS
- Express backend with JWT auth and Task CRUD
- MongoDB (Mongoose) models
- Postman collection

## Quick start
1. Start backend:
   - cd backend
   - copy .env.example to .env and set MONGODB_URI and JWT_SECRET
   - npm install
   - npm run dev

2. Start frontend:
   - cd frontend
   - npm install
   - create .env.local with VITE_API_BASE=http://localhost:5000 (if backend runs locally)
   - npm run dev
